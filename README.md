# 🎬 Netflix Clone - React 19

Тестовое веб-приложение в стиле Netflix, созданное с использованием **React 19**, **TypeScript** и **Vite**. Проект разработан на основе курса ["REACT 19 с НУЛЯ за 2 часа — ПОЛНЫЙ КУРС 2025!"](https://www.youtube.com/watch?v=UC2vnwCY4T4) в целях обучения.

## Возможности

- **Современный React 19** - использование новейших возможностей фреймворка
- **Vite** - молниеносная сборка и Hot Module Replacement (HMR)
- **Tailwind CSS** - быстрая и удобная стилизация
- **Темная и светлая тема** - переключение тем через Context API
- **Поиск фильмов** - с дебаунсингом для оптимизации
- **Избранное** - сохранение любимых фильмов
- **Комментарии** - модальное окно с комментариями к фильмам
- **Маршрутизация** - React Router для навигации
- **TypeScript** - полная типизация для безопасности кода
- **ESLint** - поддержка качества кода

## Быстрый старт

### Требования
- Node.js 18+ или Bun
- npm, yarn, pnpm или Bun для управления пакетами

### Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone <URL-репозитория>
   cd react-netflix-course
   ```

2. **Установите зависимости**
   ```bash
   npm install
   # или
   bun install
   ```

3. **Запустите приложение в режиме разработки**
   ```bash
   npm run dev
   # или
   bun run dev
   ```

   Приложение откроется по адресу `http://localhost:5173`


## Ключевые концепции

### Context API
- **SearchContext** - управление состоянием поиска фильмов
- **ThemeContext** - управление светлой/темной темой

### Пользовательские хуки
- **useSearch** - упрощает работу с поиском
- **useTheme** - удобный доступ к теме
- **useDebounce** - оптимизирует частые обновления

### Компоненты
- **MovieCard** - отображение информации о фильме
- **Modal** - модальное окно для подробной информации
- **Search** - поиск с live-фильтрацией
- **FavoriteButton** - управление избранным
